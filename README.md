WebScraper app

INSTRUCTION:
    To run the app, do the following:

        1. npm install
        2. bower install
        3. npm start


Technology stack:

        Front end:
            Angular.js
        Back end:
            Node.js, Express.js
        Testing:
            Mocha
        Doc Generator:
            ESDoc

    3rd-party libraries:

        x-ray       --Node.js module for scraping websites
        file-saver  --Angular module for saving data into files
